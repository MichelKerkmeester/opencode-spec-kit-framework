# Skill Graph Node Template

Use this template when adding a new node under `nodes/`.

---

<!-- ANCHOR:template -->
## Template

Copy the structure below into a new file and replace placeholders with real content.

```md
---
description: "One line summary of this node"
---
# Node Title

Short intro. Explain what this node covers and when to read it.

## Rules or Workflow
- Rule 1
- Rule 2
- Rule 3

## Cross References
- Add links to related nodes in real prose.
- Example syntax: \[\[nodes/related-topic\]\]
```

Checklist before saving:
- Description is clear.
- Headings are specific.
- Content is scoped to one topic.
- Links point to real files.
