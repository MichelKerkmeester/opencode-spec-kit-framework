# Template Examples

Reference-only examples for level formatting and depth.

## Overview
<!-- ANCHOR:overview -->

Examples show how content scales from Level 1 through Level 3+.
Use them for comparison, not direct copying.

<!-- /ANCHOR:overview -->

## Structure
<!-- ANCHOR:structure -->

| Example | Purpose |
|---|---|
| `level_1/` | minimal scope and lightweight planning |
| `level_2/` | verification-focused execution and checklist use |
| `level_3/` | architecture and decision-record usage |
| `level_3+/` | governance, compliance, and coordination patterns |

<!-- /ANCHOR:structure -->

## Usage
<!-- ANCHOR:usage -->

1. Choose your level from real requirements.
2. Copy templates from `../level_N/`, not from `examples/`.
3. Compare your spec docs to examples for quality and completeness.

<!-- /ANCHOR:usage -->
