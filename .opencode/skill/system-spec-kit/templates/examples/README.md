### What is Template Examples?

These examples serve as **reference materials** showing how to properly fill out templates at each level. They demonstrate progressive enhancement from Level 1 to Level 3+ using a consistent fictional scenario (user authentication).

### Key Features

| Feature | Description |
|---------|-------------|
| **Progressive enhancement** | Shows appropriate content depth for each level |
| **Consistent scenario** | User authentication example across all levels |
| **Reference only** | Learn patterns without copying directly |
