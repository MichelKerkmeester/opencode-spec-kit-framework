04 // Step-by-Step Instructions

Build Production Assets

Run the build pipeline to generate minified, versioned output files.

# From project root
npm run build:prod
npm run minify -- --version 2.4.0
npm run verify:checksums
Expected: 14 files in dist/ with matching SHA256 checksums

Upload to CDN

npx wrangler r2 object put cdn-bucket/v2.4.0/js/ \ --file dist/js/ --content-type application/javascript
npx wrangler r2 object put cdn-bucket/v2.4.0/css/ \ --file dist/css/ --content-type text/css

Purge Cache & Verify

curl -X POST "https://api.cloudflare.com/client/v4/zones/ZONE_ID/purge_cache" \ -H "Authorization: Bearer $CF_TOKEN" -d '{"purge_everything":true}'
# Verify
curl -I https://cdn.example.com/v2.4.0/js/app.min.js
Expected: HTTP 200, correct content-type, cache-control headers

Update Webflow & Publish

<!-- Head code -->
<link rel="stylesheet" href="https://cdn.example.com/v2.4.0/css/styles.min.css">
<!-- Footer code -->
<script src="https://cdn.example.com/v2.4.0/js/app.min.js" defer></script>

Tag Release

git tag -a v2.4.0 -m "Release v2.4.0: CSS consolidation, analytics migration"
git push origin v2.4.0
