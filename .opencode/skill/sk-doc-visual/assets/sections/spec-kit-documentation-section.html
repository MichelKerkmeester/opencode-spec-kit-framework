Documentation Levels

Level LOC Recommended Files Use When
1 <100 spec.md, plan.md, tasks.md, summary.md Baseline changes
2 100-499 Level 1 + checklist.md Validation required
3 >=500 Level 2 + decision-record.md Complex changes
3+ Comp 80+ Level 3 + AI protocols, extended checklist Extended governance

Template Architecture

Example template set shared across levels with reusable base and extension files.

Template Purpose
spec.md Scope, requirements, and constraints
plan.md Implementation approach and decisions
tasks.md Ordered task breakdown with status
checklist.md Validation checklist items
decision-record.md Decision log with rationale and trade-offs
research.md Research findings (optional)
implementation-summary.md Implementation summary after completion

Metadata Normalization

Example workflow for normalizing metadata across markdown files.

# 1) Dry-run metadata normalization
node scripts/dist/context/backfill-frontmatter.js --dry-run

# 2) Apply metadata normalization
node scripts/dist/context/backfill-frontmatter.js --apply

# 3) Verify composed templates
bash scripts/templates/compose.sh --verify

# 4) Rebuild embeddings index
node scripts/dist/context/reindex-embeddings.js
