# Evidence Directory

This directory stores all evidence files referenced in the checklist and implementation summary.

## Evidence Types

- **[E:filename]** - Evidence files (logs, reports, outputs)
- **[Test:testname]** - Test execution results
- **[Screenshot:filename]** - Screenshots of validations, renderings, etc.

## Organization

Evidence files should be named clearly to match their checklist references:

- `anchor-taxonomy.md` - Anchor taxonomy definition
- `migration-dryrun.log` - Dry-run migration log
- `validation-post-migration.json` - Post-migration validation report
- `coverage-post-migration.json` - Coverage validation report
- `git-diff-*.md` - Git diffs for template changes
- `backup-verification-checksums` - Backup integrity checksums
- etc.

**Status**: Will be populated during implementation phases
