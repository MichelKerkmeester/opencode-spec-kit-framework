10 // Extensibility

Adaptation & Extension

The extension model is built strictly on convention over configuration. Reusable placeholders and isolated modules allow the environment to evolve without destabilizing the core architecture.

System templates provide the underlying structure; you own the content. The platform adapts to your specific project needs.

Active Extensions

Generative capabilities injected into the agent ecosystem.

Skills

Extension Skills

Auto-detected by the routing module. Design new skills to introduce specialized analytical workflows or external tool integrations without modifying base prompts.

/create:skill my-skill
Roles

Extension Roles

Define bespoke behavior profiles and operational access limits. Roles allow you to sculpt how the system responds across different contexts or user personas.

/create:role example-role

Commands

Extension Commands

Templates

Template D Inheritance
