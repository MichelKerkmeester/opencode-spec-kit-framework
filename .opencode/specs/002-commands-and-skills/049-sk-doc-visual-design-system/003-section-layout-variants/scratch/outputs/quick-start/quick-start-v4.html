Onboarding path

Quick Start

Get your environment running locally in three executable steps. Action-oriented setup with minimal configuration.

01

Verify Prerequisites

Ensure your system meets the baseline requirements before proceeding.

  • Node.js v18.0+
    node --version
  • Git CLI
    git --version
02

Clone & Install

Pull the repository and install all required framework dependencies securely.

bash 
# Clone the template repository
git clone https://github.com/example/workspace.git
cd workspace

# Install dependencies via clean install
npm ci
03

Initialize Workspace

Start the local development server and execute the baseline health check.

bash 
# Start local development environment
npm run dev

# Target output expectation:
# ➜  Local:   http://localhost:3000/
# ➜  Network: use --host to expose
