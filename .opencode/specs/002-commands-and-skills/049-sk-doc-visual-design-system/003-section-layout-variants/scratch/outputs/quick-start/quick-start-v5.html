02 // Quick Start

Milestone Tracker

Follow this sequential path to initialize the environment. Do not proceed until each verification gate is cleared.

  1. 01. Environment Prerequisites

    Ensure base system dependencies match the minimum required versions.

    Node.js 18+ node --version
    Template CLI opencode --version
    Verification Gate

    v18.0.0 or higher is present in terminal output.

  2. 02. Base Installation

    Acquire the repository and resolve local dependencies.

    bash 
    git clone https://github.com/example/repo.git
cd repo
npm install
    Verification Gate

    npm output confirms added N packages without audit errors.

  3. 03. Local Bootstrapping

    Initialize the configuration layer and start the local development server.

    bash 
    npm run init:config
npm run dev
    Verification Gate

    Server resolves successfully at http://localhost:3000.
