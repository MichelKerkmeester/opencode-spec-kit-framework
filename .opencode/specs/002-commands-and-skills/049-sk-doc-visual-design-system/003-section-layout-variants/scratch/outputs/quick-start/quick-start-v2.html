02 // Quick Start

Prerequisites
  • Node.js 18+ node -v
  • Template CLI v1.2+ opencode --version
  • Git git --version

Lane A: New Project

Scaffold a fresh repository with recommended defaults and boilerplate architecture.

bash 
# 1. Initialize via create wizard
npx create-opencode-app@latest my-app

# 2. Enter workspace
cd my-app

# 3. Start development server
npm run dev

Lane B: Existing Project

Integrate capabilities into an already established repository without overwriting config.

bash 
# 1. Install core dependencies
npm install @opencode/core @opencode/cli

# 2. Generate local config
npx opencode init --minimal

# 3. Verify integration
npx opencode doctor
