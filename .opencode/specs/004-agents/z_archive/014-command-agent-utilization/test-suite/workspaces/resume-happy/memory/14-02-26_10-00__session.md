## Key Decisions

### JWT Signing Algorithm
- **Choice**: RS256 (RSA + SHA-256)
- **Rationale**: Asymmetric keys allow token verification without exposing signing key
- **Alternatives rejected**: HS256 (shared secret, less secure for distributed systems)

### Token Expiry
- **Access token**: 15 minutes
- **Refresh token**: 7 days
- **Rationale**: Industry standard balance of security vs UX
