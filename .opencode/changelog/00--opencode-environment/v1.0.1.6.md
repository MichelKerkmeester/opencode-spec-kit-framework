## [**1.0.1.6**] - 2025-12-29

**Spec Kit Memory System Upgrade** — Major release with 7 new tools, checkpoint support, and 30+ metadata fields for advanced filtering.

---

## Highlights

### New

- **7 New MCP Tools**: `memory_health`, `memory_delete`, `memory_update`, `memory_validate`, `memory_index_scan`, `checkpoint_create`, `checkpoint_list`, `checkpoint_delete`
- **30+ Metadata Fields** — `specFolder`, `level`, `status`, `tags[]`, `priority`, `git_branch`, `author`, `related_specs[]`, `dependencies[]`, `blockers[]`
- **Smart Ranking** — Time-decay scoring with configurable weights
- **Auto-Anchors** — Automatic extraction of heading-based anchors for targeted queries
- **Embedding Providers** — Support for Voyage AI (1024d) and OpenAI (1536d)
- **Multi-Concept Search** — Search by multiple concepts (OR logic)
- **Advanced Filters** — By spec folder, level, status, tags, priority, date ranges
- **Trigger Phrase Limits** — Configurable max per memory (prevents bloat)

### Changed

- **Database schema** updated with 30+ new columns
- **Memory index** now includes heading-based anchors
- **Constitutional memories** separated from specs
- **Smart scoring** replaces pure semantic matching
- **`generate-context.js`** outputs all 30+ metadata fields

### Fixed

- **Migration scripts** for schema changes (v15 → v16)
- **Backward compatibility** for existing memories
- **Error handling** for missing metadata fields
