## [**1.0.1.2**] - 2025-12-29

**Project-Local Indexing** — Adds per-project Narsil index support for isolated codebase indexing and team collaboration.

---

## Highlights

### Local Index Management

- **Project-local indexing** — `.narsil-index/` directory instead of shared `~/.cache/narsil-mcp/`
- **Persistence control** — `--persist` flag for index persistence across sessions
- **Custom paths** — `--index-path` option for index location flexibility
- **Manual save operations** — `save_index` tool for explicit index persistence
- **HTTP server mode** — Full documentation for standalone deployment

## Files Changed

- Narsil configuration (project-local paths)
- Index management tools and CLI
- HTTP server setup documentation

## Upgrade

Optional — Projects can opt-in to local indexing:
```bash
narsil-mcp --persist --index-path .narsil-index/
```
