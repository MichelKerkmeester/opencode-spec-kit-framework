## Highlights

### Breaking Changes

- **LEANN completely removed** — Migration path required
- **API changes** — `leann_leann_search()` → `narsil.narsil_neural_search()`
- **MCP transition** — LEANN MCP → Narsil (via Code Mode)
- **Skill updates** — `mcp-leann` skill → `mcp-narsil` skill
- **Embedding backends** — MLX embeddings → Voyage/OpenAI/ONNX backends
- **Skill count** — Reduced from 8 to 7 (consolidated Narsil tooling)

### New Capabilities

- **76 specialized tools** — Neural search, security scanning, call graph analysis
- **Unified code intelligence** — Single interface for all search patterns
- **Multiple embedding backends** — Flexible embedding strategy

## Files Changed

- MCP configuration (Narsil integration)
- Skill definitions and routing
- Search API implementations
- Backend embedding configuration

## Upgrade

**BREAKING CHANGE** — Manual migration required:
1. Update `opencode.json` to remove LEANN
2. Update skills from `mcp-leann` → `mcp-narsil`
3. Update search calls: `leann_leann_search()` → `narsil.narsil_neural_search()`
4. Test search functionality in your project
