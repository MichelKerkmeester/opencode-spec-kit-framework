## Highlights

### New

- **`validate_document.py`** — Automated README/documentation format validator with TOC format, H2 emojis, required sections validation; exit codes 0/1/2; JSON output mode; auto-fix capability (`--fix`, `--dry-run`); document type detection
- **ALL CAPS validation** — Section names must be uppercase with new error types `h2_not_uppercase` and `toc_not_uppercase`, auto-fixable with `--fix` flag
- **Path exclusions** — Auto-skip validation for auto-generated and third-party files with `--no-exclude` flag override
- **`template_rules.json`** — Machine-readable template specifications with 5 document types, severity levels, auto-fix rules
- **Test suite** — 6 test files covering all error types; results: 6/6 pass, batch: 53 VALID, 19 SKIPPED, 0 INVALID

### Changed

- **`write.md` agent** — Added VALIDATE FORMAT gate (step 8)
- **SKILL.md** — Added ALWAYS rule #9 for validate_document.py
