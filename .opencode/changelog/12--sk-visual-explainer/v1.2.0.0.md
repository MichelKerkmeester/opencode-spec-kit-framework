## Highlights

### New

- **Single visual command entrypoint** - Added `/create:visual_html` to consolidate visual generation, plan review, diff review, recap, and fact-check.
- **Dual workflow assets** - Added `.opencode/command/create/assets/create_visual_html_auto.yaml` and `.opencode/command/create/assets/create_visual_html_confirm.yaml`.
- **General-agent verification** - New command uses `@general` Phase 0 verification and explicitly avoids `@write` dependency.

### Changed

- **Create command catalog expanded** - Updated create command docs to include `visual_html` command structure, usage, and troubleshooting.
- **Skill activation mapping updated** - Updated `sk-visual-explainer` command references and command-boost mapping to `create:visual_html` mode routing.
- **Quick reference command cheat sheet updated** - Replaced legacy `/visual-explainer:*` invocations with `/create:visual_html --mode ...` variants.
- **Spec folder renamed** - Renamed `.opencode/specs/002-commands-and-skills/042-sk-visual-explainer-speckit-alignmnet` to `.opencode/specs/002-commands-and-skills/042-sk-visual-explainer-refinement`.

### Removed

  - `.opencode/command/visual_explainer/plan-review.md`
  - `.opencode/command/visual_explainer/diff-review.md`
