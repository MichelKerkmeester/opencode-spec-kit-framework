### Critical Gaps Identified

1. **No Node-to-Node LINKS_TO edges** -- 633 edges are structural (Skill->Index->Node hierarchy), not semantic cross-references. All traversal queries between Node-labeled nodes return 0 rows.
2. **Case-sensitive CONTAINS** -- `"commit"` misses `"Commit"`, `"conventional"` misses `"Conventional"`. ~30% of valid queries are invisible due to capitalization.
3. **12 zero-hit domain terms** -- css, layout, network, API, accessibility, ARIA, audit, deployment, handler, export, tool chain, conflict
4. **skill_advisor.py at 45% accuracy** -- `!network` scored as negative signal for DevTools (inverted), uncertainty threshold too conservative
