## Highlights

### New

- **Stack detection via marker files** — Automatic detection: `go.mod` → Go, `package.json` with "express" → Node.js, `next.config.js`/react → React, `app.json` with "expo" → React Native, `Package.swift` → Swift
- **Hierarchical resource structure** — `references/{category}/{stack}/` and `assets/{category}/{stack}/`
- **Smart resource routing** — Python pseudocode router with STACK_MARKERS, STACK_FOLDERS, TASK_KEYWORDS, LOAD_LEVELS, `classify_task()`, `route_resources()`
- **3-Phase lifecycle** — Phase 1: Implementation, Phase 2: Testing/Debugging, Phase 3: Verification
- **Stack-specific verification commands** — Go, Node.js, React, React Native, Swift each with test/lint/build
- **References (36 files)** — Standards across all 5 stacks
- **Assets (42 files)** — 15 checklists + 27 pattern files
